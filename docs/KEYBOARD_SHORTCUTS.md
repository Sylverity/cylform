# Keyboard shortcuts

Cylform shortcuts are configurable in **Settings -> Shortcuts**. On macOS, `CmdOrCtrl` displays as `Cmd`; on Windows and Linux, it displays as `Ctrl`.

## File

| Action | Default shortcut |
| --- | --- |
| Open File | `CmdOrCtrl+O` |
| Open Recent | `CmdOrCtrl+Shift+O` |
| Close Current Molecule | `CmdOrCtrl+W` |
| Export Figure | `CmdOrCtrl+E` |
| Previous Tab | `CmdOrCtrl+PageUp` |
| Next Tab | `CmdOrCtrl+PageDown` |
| Previous File | `Alt+Left` |
| Next File | `Alt+Right` |

## Navigation

| Action | Default shortcut |
| --- | --- |
| Reset View | `R` |
| Clear Selection | `Escape` |
| Front View | `1` |
| Top View | `2` |
| Right View | `3` |
| Iso View | `4` |

Mouse navigation uses left-drag to rotate, right-drag to pan, and scroll to zoom.

## Selection and Visibility

| Action | Default shortcut |
| --- | --- |
| View Mode | `V` |
| Measure Mode | `M` |
| Atom Selection | `A` |
| Bond Selection | `B` |
| Atom+Bond Selection | `Z` |
| Label Mode | `L` |
| Cycle Hydrogen Visibility | `H` |

`H` only cycles hydrogen visibility: shown, hidden, then hide C-H. Render profiles, including CYLview, Ball and stick, and Houkmol, are changed from the View overlay or Settings and do not have default letter shortcuts.

## Help

| Action | Default shortcut |
| --- | --- |
| Settings | `CmdOrCtrl+,` |
| Keyboard Shortcuts | `Shift+/` |

Shortcuts are ignored while dialogs are open except `Escape`, which closes the topmost dialog. Shortcuts are also ignored while typing in text inputs, editing labels, or using form controls.
